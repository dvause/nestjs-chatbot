
## Description

An [OpenAI](https://openai.com) chatbot written in the [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Configuration
```dotenv
OPENAI_API_KEY='[your OpenAI API key]'
SYSTEM_PROMPT='You are a helpful chatbot that speaks in a Medeval dialect.'
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
